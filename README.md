This repository contains SVG recreations of the low-resolution PNG icons and radar traces in EmptyEpsilon.

Original artwork credits:

- Ship radar traces: [Jadael](https://github.com/Jadael)
- UI icons and elements: Interesting John, Fouindor

Use the `export-svgs-to-pngs.sh` script to convert all SVGs in a directory to PNGs. Requires Inkscape for conversions, ImageMagick for sizing, and pngcrush for output optimization. Works best when the SVG source has page size metadata in pixel units.
